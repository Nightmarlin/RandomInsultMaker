# RandomInsultMaker
Randomly generates an insult from a list. May or may not work

## How to use
> 1) Require the module
> 2) Call the module to a variable
> 3) Enjoy the wrath of insults

### Parameters
> RandomInsult() will return
```
"You [ADJ] [ADJ] [NOUN]!"
```
> RandomInsult(X [integer]) will return
```
"You {X [ADJ]s} [NOUN]!"
```
